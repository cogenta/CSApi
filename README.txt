CSApi
=====

CSApi is a library for using the Cogenta Shopping API with iOS. The Cogenta Shopping API is a web service intended to give third parties access to Cogenta's [retail and pricing intelligence data](http://www.cogenta.com/services/mobile-shopping/).

Getting Access
--------------

You'll need credentials to use the Cogenta Shopping API. [Give us a call or email us](http://www.cogenta.com/contact/) to get a developer account.

Installation
------------

[CocoaPods](http://cocoapods.org/) is the easiest way to use CSApi.

    pod 'CSApi', :git => 'https://github.com/cogenta/CSApi.git'
